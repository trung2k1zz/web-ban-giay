Active Products Tables - PROTAS - is the WooCommerce Products Table plugin displaying shop products in table format

== Description ==

**Active Products Tables for WooCommerce** - PROTAS (second name is WOOT) - is WordPress plugin for WooCommerce products, created for displaying woo shop products in the table format. Woo Products tables makes focus for your buyers on the things they want to get, nothing superfluous, just what the client wants, and full attention to what is offered! Ideal way to higher sells on your woocommerce store!

Latest PHP 8.x - FULL COMPATIBILITY! Compatibility: from v.3.6 to the latest woocommerce version.

Demo site: [demo.products-tables.com](https://demo.products-tables.com/).


If you need plugin for pages, posts and their custom post types use: [TableOn](https://wordpress.org/plugins/posts-table-filterable/).


### Active Products Tables Features:

&#9989;&nbsp;Columns constructor

&#9989;&nbsp;Neat shortcode [[woot]](https://products-tables.com/shortcode/woot/) with heap of attributes for flexibility

&#9989;&nbsp;[Many possible tricks](https://products-tables.com/document/after-woot-installation-you-can-do-next/) with the products tables!

&#9989;&nbsp;[More than 40 predefined column fields](https://demo.products-tables.com/woot-with-all-possible-columns/) is possible to display

&#9989;&nbsp;[Possibility to create custom columns](https://products-tables.com/how-to-add-custom-column-to-the-tables/)

&#9989;&nbsp;Skins for each table. On the same web page can be represented [some products tables with the different skins](https://demo.products-tables.com/different-skins/). Create [your own skins by CSS](https://products-tables.com/document/skins/).

&#9989;&nbsp;Powerful product filter constructor

&#9989;&nbsp;[Big variety of filter elements](https://demo.products-tables.com/woot-with-all-possible-filters/) by: price, title, content, excerpt, SKU, attributes, taxonomy, meta fields, in stock, on sale, width, height, length, weight, etc...

&#9989;&nbsp;Filters can be represented as: dropdown, multi drop-down, range sliders, textinput, calendars, switcher

&#9989;&nbsp;Predefinition mechanism: show to your customers predefined and relevant set of products as in the table, so in the popup.

&#9989;&nbsp;25 ways of the products sorting

&#9989;&nbsp;Power feature as [remote tables](https://inbuilt.products-tables.com/) which gives ability to spread your shop products to any another sites even created on pure HTML without any CMS, what allows you get more sells and also realize your own referral program

&#9989;&nbsp;Column "Gallery" - smooth CSS gallery to showcase the best of your products. Also a product gallery can be represented by shortcode [[woot gallery]](https://products-tables.com/shortcode/woot_gallery/)

&#9989;&nbsp;Shortcode [[woot_cart]](https://products-tables.com/shortcode/woot_cart/) allows to create custom cart page

&#9989;&nbsp;Shortcode [[woot_reviews]](https://products-tables.com/shortcode/woot_reviews/) allows to show the table with reviews for the current product on its single page or any another one

&#9989;&nbsp;Shortcode [[woot_cross_sells]](https://products-tables.com/shortcode/woot_cross_sells/) allows to show the table with cross sell products for the current product on its single page or any another one

&#9989;&nbsp;Shortcode [[woot_upsells]](https://products-tables.com/shortcode/woot_upsells/) allows to show the table with upsell products for the current product on its single page or any another one

&#9989;&nbsp;Shortcode [[woot_variations]](https://products-tables.com/shortcode/woot_variations/) allows to show the table with variations products for the current product on its single page or any another one

&#9989;&nbsp;Shortcode [[woot_related]](https://products-tables.com/shortcode/woot_related/) allows to show the table with related products for the current product on its single page or any another one

&#9989;&nbsp;Shortcode [[woot_favourites]](https://products-tables.com/shortcode/woot_favourites/) allows to show the table with selected products by the current user

&#9989;&nbsp;Shortcode [[woot_single]](https://products-tables.com/shortcode/woot_single/) allows to show the table with a product parameters for any one selected ones

&#9989;&nbsp;Ability to show multi vendors their products using attribute param author: [[woot author=23]](https://products-tables.com/shortcode/woot/)

&#9989;&nbsp;Ability to create custom columns, for example such as ["Ask manager about the product"](https://products-tables.com/shortcode/woot_popup_iframe_button/) using any contact form plugins

&#9989;&nbsp;Responsivity: all the products tables have an option for compact view on mobile devices!

&#9989;&nbsp;Big and flexible set of options for each product table, some of them: Hide added in cart products, Show cart, Show Sorting Dropdown, Sorting Dropdown Fields, Compact view width, Use load more button, Hide filter form, Show print button, Default order by, Per page drop-down position, Per page drop-down position, Per page values, Per page default

&#9989;&nbsp;Set of options for each product column, some basic: Width, Font size, Font family, Color, Background, Hide on small screen

&#9989;&nbsp;Ability to add meta fields data in the table columns using in-built constructor

&#9989;&nbsp;Ability to [show a product table in the popup on click](https://demo.products-tables.com/table-as-link/). Show tables in popup in your shop text content with relevant predefined tables. Unlimited count of tables in popup per page can be created!

&#9989;&nbsp;In-built shopping cart on top of the tables, which also presented as shortcode and can be pasted on the site pages

&#9989;&nbsp;[Caching table of products](https://demo.products-tables.com/cached-table-of-products/) mechanism

&#9989;&nbsp;Print button

&#9989;&nbsp;Possibility to add to cart many products in one click

&#9989;&nbsp;Mobile compact view option

&#9989;&nbsp;CSS editor

&#9989;&nbsp;Pagination

&#9989;&nbsp;[Compatible with WOOF](https://demo.products-filter.com/demonstration-of-woot-and-woof-compatibility)

&#9989;&nbsp;[Compatible with MDTF](https://woocommerce.wp-filter.com/demonstration-of-woot-and-mdtf-compatibility/)

&#9989;&nbsp;[Compatible with WOOCS](https://demo.currency-switcher.com/demonstration-of-woot-and-woocs-compatibility/)

&#9989;&nbsp;[Compatible with BEAR](https://bulk-editor.com/)

&#9989;&nbsp;Compatible with [Custom Post Type UI](https://wordpress.org/plugins/custom-post-type-ui/)

&#9989;&nbsp;WPML compatible

&#9989;&nbsp;Shortcodes works in Elementor and Page builder page content

&#9989;&nbsp;[Power and strong API](https://products-tables.com/codex/)

&#9989;&nbsp;no jQuery - 100% pure JavaScript
